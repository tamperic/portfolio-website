Meet App

The Meet App is a serverless Progressive Web Application (PWA) built with React, based on the Test-Driven Development (TDD), and using the Google Calendar API to fetch upcoming events.

Key Features:

  • filter events by city,
  • specify number of events,
  • show/hide event details,
  • use the app when offline,
  • add an app shortcut to the home screen,
  • display charts visualizing event details.

Technologies Used:

  • React library for building user interfaces,
  • Google API Calendar for fetching upcoming events,
  • TDD (Test-Drive Development) - unit and integration testing,
  • (Behavior-Driven Development) - acceptnace testing easy to read and find even by non-developer stakeholders,
  • OAuth2 for user authentication,
  • Jest - JavaScript testing framework,
  • Jest-Cucumber - writing Jest tests using Gherkin syntax (Behavior-Driven Development),
  • AWS Lambda for serverless backend - no backend maintenance and using a traditional server,
  • Atatus - online performance monitoring tool,
  • Deployed on GitHub Pages.
