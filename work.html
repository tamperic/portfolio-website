Meet App

The Meet App is a serverless Progressive Web Application (PWA) built with React, based on the Test-Driven Development (TDD), and using the Google Calendar API to fetch upcoming events.

Users will be able:

  • to search for events,
  • to search for events,
  • to filter them by city,
  • to specify number of events,
  • to view more or less details about one specific event.

Buzzing Chat App

The Buzzing Chat App is a simple and user-friendly chat application that allows users to easily connect with friends, family, and others. Users can share messages, images, and their location, and personalize their chat experience by choosing one of four given background colors.
