Meet App

Buzzing Chat App

The Buzzing Chat App is a simple and user-friendly chat application that allows users to easily connect with friends, family, and others. Users can share messages, images, and their location, and personalize their chat experience by choosing one of four given background colors.

Tech Stack:

  • the app is written in React Native.
  • the app is developed using Expo.
  • chat conversations are stored in Google Firestore Database, and users will be anonymously authenticated also via Google Firestore Database.
  • chat conversations are also stored on local storage.
  • images are stored in Firebase Cloud Storage.
  • the chat interface and functionality are created using the Gifted Chat library.

MovieFetcher - Movie API

This is the backend project for a movie database API built with Node.js, Express, and MongoDB. It allows users to create or delete an account, log in, view list of all movies, get detailed information about movies, directors, genres, as well as to manage their favorite movies.

Tech Stack:

  • Node.js.
  • MongoDB with Mongoose - a NoSQL database used to store user and movie data, with Mongoose providing a schema-based modeling interface.
  • Middlwares: Morgan - a logging middleware that outputs HTTP request details to the console for easy monitoring and debugging, bodyParser and Express.js - middlewares used to parse incoming request bodies (JSON and form data) so the server can access client input.
  • Passport.js (Local and JWT Strategies) - handle user authentication securely.
  • Bcrypt - a library used to hash user passwords before storing them in the database.
  • Express-Validator - for an user's input validation.
  • JSDoc - for a clean documentation from code comments.
  • Postman - a tool used to test and visualize HTTP requests and responses during API development.
  • Heroku - a cloud platform used to deploy and host the backend server so it's accessible online.
See project on GitHub

MovieFetcher-client React App

MovieFetcher-client React App is a client-side single page application (SPA) built with React that allows users to explore a wide selection of movies and interact with a RESTful API backend. It uses a component-based architecture built in React to create a dynamic user experience.

Key Features:

  • Sign Up & Login,
  • View a list of all available movies,
  • Possibility to search movies by title, genre or director's name.
  • See movie details (title, image, description, genre, director, actors, release year, rating, duration) and similar movies by gerne
  • Add or remove any movie from your personal list of favorites.
  • Manage user profile - view user details, detele account, log out, view list of favorite movies.

Tech Stack:

  • React.js
  • React-router - (a state-based) library for routing between multiple views.
  • React-Bootstrap - a popular frontend framework that provides a responsive and consistent UI.
  • Parcel bundler - a build tool for faster performance during development process
  • Prop-Types - a runtime type checking for React props.
